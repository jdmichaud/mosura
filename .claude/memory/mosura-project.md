- The Ghidra MCP server (GhidraMCP headless v4.3.0, port 8089) backs **Ghidra 12.0.3 PUBLIC** at `/home/jd/tools/ghidra_12.0.3_PUBLIC/`.

**Test-baseline strategy** (full design in `mosura/docs/testing-baseline.md`): Ghidra-as-golden-oracle / characterization testing — do NOT port Ghidra's JUnit suite.
